## BEM Methodology

This project uses a **flat BEM file structure**, improving readability and long-term maintainability.

**BEM structure:**

* **Block** вҖ“ standalone components (e.g. `menu`, `about`, `footer`)
* **Element** вҖ“ parts of a block (e.g. `menu__item`, `footer__link`)
* **Modifier** вҖ“ variations (e.g. `menu__item_highlighted`)

Each block has its own CSS file, following scalable frontend styling best practices.
